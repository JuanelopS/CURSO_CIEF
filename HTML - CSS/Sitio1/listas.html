Definición del término
URL: Localizador Universal de Recursos.
Es la dirección web donde se encuentra un recurso (documento html, imagen, video)
Hiperenlace
También "enlace", "vínculo", "link", "Hipertexto" si es un texto. Es un texto o imagen que sobre el que al pulsar nos dirige a otro documento.
